Sports App Design Concept

SCREEN LIBRARY — FILE INDEX

Final drafts, split one file per feature area so each can be pulled into Claude Code independently. Every screen is an Android device frame; all files share one theme and two dependencies.

HOME FEED
Stacked hero cards — one per favorite, detail scales with game state.
HomeFeed.dc.html
LIVE GAME DETAIL
Per-sport live screens — football, basketball, cricket, olympics, soccer, tennis, hockey, golf. Shared card grammar, sport-specific situation card.
LiveGame.dc.html
EVENT LIST
Play-by-play / timeline tab as a full screen, one grammar across sports.
EventList.dc.html
BOX SCORES
Dense stat tables — box score, game leaders, season stats, line score.
BoxScore.dc.html
TEAM & PLAYER
Overview pages across four sports + player page. Logo / headshot image slots.
TeamPlayer.dc.html
TOURNAMENTS
Four bracket grammars — World Cup groups, Wimbledon draw, March Madness, College World Series.
Tournaments.dc.html
F1 CIRCUIT
Circuit tab — track map, venue fact grid, lap record card. Spa-Francorchamps.
Circuit.dc.html
VENUE
Venue tab — photo slot, capacity fact grid, game-day conditions. Wrigley Field.
Venue.dc.html
STANDINGS & FAVORITES
Division table with playoff cut line, long-press follow sheet, manage-following tab.
Standings.dc.html
