How to read the table:
- **presence** = % of responses (or parent objects) that carry the field. Low % usually means sport-specific or situational.
- **sports** = which sports the field was observed in (`all` = every sport crawled for this endpoint).
- `[]` in a path = array element; `{key}` = a dictionary whose keys are ids (play ids, atBat ids). `str-numeric` = string that always holds a number (ESPN serves scores as strings).
- **values** exhaustively lists small closed sets (enums) with observation counts; **e.g.** shows samples of open sets.
